# JunoWrapper

A simple wrapper to have a better interaction to the [Juno](https://www.juno.com.br/) API.

## Installation

Just add `juno_wrapper` to your dependencies list on `mix.exs`:

```elixir
def deps do
  [
    {:juno_wrapper, "~> 0.1.0"}
  ]
end
```
